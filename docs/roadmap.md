# Trellis Roadmap

## v0.1 (current)

- local-first filesystem registry
- install/remove/list/search/info/update/doctor commands
- package spec validation
- receipts and deterministic cellar paths
- checksum groundwork

## v0.2

- lockfile and deterministic dependency resolution
- stronger checksum verification modes
- atomic install transactions and rollback hooks
- richer doctor diagnostics

## v0.3+

- multiple registries/taps
- remote registry sync and publish flow
- signature verification and trust policy engine
- compatibility and migration tooling
